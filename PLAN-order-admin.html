3Thành phần & thao tác

📄 Màn chi tiết — hiển thị

  • Khách + SĐT + địa chỉ
  • Nguồn đơn (tự lấy từ data)
  • Ngày nhận/trả/số ngày
  • Cọc & phí tạm giữ
  • Sản phẩm + ảnh + tổng tiền

⚡ Màn chi tiết — one-click

  • Gọi / Zalo / Copy SĐT
  • Đánh dấu cọc/phí đã thu (AJAX)
  • Trạng thái đơn (WC) = radio (lưu ngay)
  • Trạng thái giao nhận = radio (lưu ngay)
  • In phiếu / Copy đơn

➕ Màn tạo đơn

  • Tìm khách theo SĐT / nhập mới
  • Tìm & thêm sản phẩm + SL
  • Ngày nhận/trả → tự tính số ngày + tiền
  • Cọc / phí / ship / giảm / ghi chú
  • Tạo đơn 1 chạm → mở luôn màn chi tiết

4Cấu trúc file sẽ tạo

inc/admin-order/
├── index.php              # bootstrap: hook, enqueue, helper screen HPOS
├── detail.php             # metabox chi tiết + AJAX cập nhật trạng thái/cọc + in phiếu
├── create.php             # submenu "Tạo đơn thuê" + AJAX tìm khách + tạo đơn
├── order-builder.php      # okhub_build_rental_order() — dùng chung cho tạo tay & import
├── views/
│   ├── panel.php          # markup panel chi tiết
│   ├── create-form.php    # markup form tạo đơn
│   └── print.php          # phiếu in gọn
└── assets/
    ├── detail.css / detail.js     # radio/toggle → AJAX, call/copy/print
    └── create.css / create.js     # tìm SP/khách, tự tính tiền theo ngày

functions.php  →  require get_theme_file_path('/inc/admin-order/index.php');

Hook chính — Chi tiết: add_meta_boxes (thêm panel + ẩn box thừa), wp_ajax_okhub_update_order_meta, wp_ajax_okhub_print_order. Tạo đơn: admin_menu (submenu dưới WooCommerce), wp_ajax_okhub_lookup_customer (tìm khách theo SĐT từ đơn cũ), dùng lại WooCommerce product search (woocommerce_json_search_products_and_variations), wp_ajax_okhub_create_rental_order (tạo đơn → trả URL chi tiết để JS redirect). Tất cả có nonce + kiểm tra manage_woocommerce + whitelist giá trị.

Giá theo ngày & meta thuê (_rental_start_date, _rental_end_date, _rental_days, _rental_status, _deposit_*, _collateral_*) tái dùng đúng schema của import-order.php để màn chi tiết & đơn import nhất quán.

5Kiểm thử

  • Chi tiết: mở 1 đơn đã import → panel trên cùng đủ thông tin; radio trạng thái + toggle cọc lưu ngay (có order note); In/Copy/Gọi/Zalo chạy; box thừa đã ẩn/thu gọn; nút Cập nhật vẫn lưu.
  • Tạo đơn: vào WooCommerce → Tạo đơn thuê; gõ SĐT khách cũ → điền sẵn; thêm sản phẩm; đổi ngày/SL → tiền tự tính; bấm Tạo đơn → tạo WC order đúng tổng + meta thuê, rồi mở thẳng màn chi tiết.
  • Đối chiếu số tiền với công thức đơn giá × SL × số ngày (giống import-order.php).
